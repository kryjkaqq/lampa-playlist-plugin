Плагин для медиацентра Lampa, который изменяет логику передачи контента во внешние плееры.
Что делает

Вместо передачи ссылки на одну выбранную серию, плагин формирует и отправляет во внешний плеер плейлист со всеми доступными сериями. Это позволяет внешнему плееру автоматически воспроизводить сериал по порядку, переключая серии без участия пользователя.
